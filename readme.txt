本文件夹内保存了 SR620数据记录器器 的源代码
用于保存SR620的测量数据。

程序的依赖各项环境如下：
Win10 x64 1909; Visual Stdio 2019; Python 3.7.5; altgraph 0.17; Click 7.0; cycler 0.10.0; future 0.18.2; kiwisolver 1.1.0; matplotlib 3.1.3; numpy 1.18.1; opencv-python 4.2.0.32; packaging 20.1; pefile 2019.4.18; pip 20.0.2; PyInstaller 3.6; pyparsing 2.4.6; PyQt5 5.14.1; PyQt5-sip 12.7.1; pyqt5-tools 5.13.0.1.5; pyqtgraph 0.10.0; pyserial 3.4; python-dateutil 2.8.1; python-dotenv 0.11.0; pywin32-ctypes 0.2.0; setuptools 45.2.0; six 1.14.0;


如果程序运行提示缺少.dll文件，请先安装VC_redist.x64.exe
